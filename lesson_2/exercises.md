1- What are the required components of an HTTP request? What are the additional optional components?

The required components are the HTTP Method (such as GET or POST) and the Path (such as /tasks). The additional optional components are Parameters, Headers, and Body.

2 - What are the required components of an HTTP response? What are the additional optional components?

The required component is the Status (such as 200 OK). The optional components are the Headers (such as ContentType) and the Body.

3 - What determines whether a request should use GET or POST as its HTTP method?

A GET request is fetching information from the server, while a POST request sends information from the client to the server.
